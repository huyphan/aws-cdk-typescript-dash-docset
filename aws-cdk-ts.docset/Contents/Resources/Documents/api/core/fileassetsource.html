Interface FileAssetSource

Represents the source for a file asset.

Synopsis

Properties

fileName

The path, relative to the root of the cloud assembly, in which this asset source resides. This can be a path to a file or a directory, dependning on the packaging type.
packaging

Which type of packaging to perform.
sourceHash

A hash on the content source. This hash is used to uniquely identify this asset throughout the system. If this value doesn't change, the asset will not be rebuilt or republished.
